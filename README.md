# California Mental Health Services KPI Dashboard

An end-to-end data pipeline and interactive Tableau dashboard analyzing mental health service utilization across California counties using real data from the California Health and Human Services (CHHS) Open Data Portal.

## Project Overview

This project explores patterns in California's public mental health system across four fiscal years (2019/20 to 2022/23), covering county-level beneficiary counts, delivery system trends, and demographic breakdowns by sex and race/ethnicity.

**Key questions answered:**
- Which counties serve the most mental health beneficiaries?
- How has service utilization grown over time across delivery systems?
- How are beneficiaries distributed by sex and race/ethnicity?

## Dashboard Views

| View | Description |
|------|-------------|
| Beneficiaries by County | Ranked bar chart of total beneficiaries across all CA counties |
| Service Trends Over Time | Line chart comparing NSMHS vs SMHS growth from 2019/20 to 2022/23 |
| Demographics by Sex | Beneficiary breakdown by sex (Female vs Male) |
| Demographics by Race | Beneficiary distribution across race/ethnicity groups |

## Key Findings

- Los Angeles County accounts for the largest share of mental health beneficiaries by a significant margin
- NSMHS (county-operated) services grew steadily while SMHS (state hospital) utilization remained flat
- Female beneficiaries outnumber male beneficiaries across all years
- Hispanic and White populations represent the two largest groups served

## Data Source

California Health and Human Services (CHHS) Open Data Portal
- Behavioral Health Program Performance Data
- Fiscal years 2019/20 through 2022/23
- [data.chhs.ca.gov](https://data.chhs.ca.gov)
